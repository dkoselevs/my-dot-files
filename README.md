Installation:
1.```git clone https://github.com/dkoselevs/my-dot-files```
2. ```cd my-dot-files```
3. ```chmod +x setup.sh```
4. ```./setup.sh```
5. Go through setup
6. Launch nvim and execute `:PlugInstall`
7. Reboot

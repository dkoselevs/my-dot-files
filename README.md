1. Install arch or arch-based system(Tested on arco linux)
2. Install paru
3. git clone this repo
4. chmod +x setup.sh
5. ./setup.sh
6. nvim; :PlugInstall
7. chsh -s /bin/zsh
8. reboot 
9. Ready to go!
